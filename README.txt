Ticket Service Homework

/*Technology Used*/
Programming Language: Java
JRE: 1.7
Build Tool: Maven
IDE: Eclipse

/*How to run*/
Run as a Java Application

/*How the application works*/
Arguments to the application have been added in the TicketSystemDemo class. It accepts the number of seats to be reserved, email of the customer and the desired Seat level in the auditorium.
The seats have been added in place of a database.

/*Things that can be improved*/
1. User input for the ticket system.
2. Ability to change the seats if the user doesn't like them.
